# Bracket Push

Given a string containing brackets `[]`, braces `{}` and parentheses `()`,
verify that all the pairs are matched and nested correctly.

# Running the tests

You can run all the tests for an exercise by entering

```sh
$ gradle test
```

in your terminal.
